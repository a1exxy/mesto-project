# Проект: Место

## Описание
Учебный проект интерактивной страницы, куда можно добавлять фотографии, удалять их и ставить лайки.

## Ссылка на проект
Проект доступен для просмотра на [GitHub Pages](https://a1exxy.github.io/mesto/)

## Используемые технологии
* HTML
* CSS

## Разрешенния экранов
от 320px до 1280px
